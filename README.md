# anima-pet-nestjs

## Description

Repositorio criado para o trabalho de api e testes da cadeira de Práticas de Engenharia de Software - Uniritter

## Instalação

```bash
$ npm install
```

## Para rodar a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
